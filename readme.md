## Starter for simple js project

Includes:

+ webpack setup for css
+ postcss
+ tailwind css

How to run?

first: `npm install`

then `npm run build` or `npm run dev`

and visit /dist/index.html

### Important notes:

contains `CleanWebpackPlugin` which deletes content of /dist folder
